A well-organized repository containing personal development environment configurations designed to streamline setup across different platforms. This collection includes platform-specific setup guides, container definitions, and AI integration configurations.

The macOS setup guide provides automated installation and configuration for:
- **Package Management:** Homebrew for dependency management
- **Window Management:** AeroSpace for efficient workspace organization
- **Development Runtimes:** Node.js via NVM, Python via uv
- **Essential Tools:** Git, GitHub CLI, usql for database management
- **Modern Package Managers:** PNPM and Bun for faster JavaScript workflows
- **AI Integration:** Claude Code CLI for AI-assisted development

The repository structure separates platform-specific setups, container configurations, and AI/MCP integrations, making it easy to navigate and adopt specific configurations as needed. All configurations are MIT licensed and designed to be forked and customized for individual developer needs.
